      <span style="font-weight:bold;">MOABB.</span> During my career, I have put great effort into popularizing a reproducible methodology for research in brain-computer interfaces (BCI). One particularly relevant contribution has been my participation in the development of the <a style="color:#FA2742; font-weight:bold;" href="http://moabb.neurotechx.com/docs/index.html" target="_blank">MOABB</a> framework, which provides the means for easily comparing classification methods on a benchmark of publicly available BCI datasets. I have contributed to its code base with the inclusion of the functionality for handling BCI data in the P300 paradigm. I have also co-organized a one-day workshop at the Graz BCI 2019 conference with a hands-on tutorial to MOABB. The code and slides for the workshop are available <a style="color:#FA2742; font-weight:bold;" href="https://github.com/plcrodrigues/Workshop-MOABB-BCI-Graz-2019" target="_blank">here</a>.
